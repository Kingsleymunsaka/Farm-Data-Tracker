# Farm Data Tracker

Author: Kingsley Munsaka

## Description
This Python project helps farmers estimate crop yields based on crop type and farm area.

## Features
- Accepts multiple crops
- Calculates estimated yield per hectare
- Displays a farm summary

## Technologies Used
- Python
- Dictionaries
- Loops
- Conditional statements

## Purpose
This project demonstrates my early programming skills and my interest in using technology to improve agriculture and food security in Zambia.
